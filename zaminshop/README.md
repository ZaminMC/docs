# ZaminShop

This wiki reflects **ZaminShop 1.24.3**.

ZaminShop is a premium shop plugin for Minecraft servers.

It lets you create GUI shops using YAML files, with support for shop packs, category menus, buy and sell flows, amount selectors, search, multiple currencies, custom item plugins, PlaceholderAPI, safety checks, sell limits, and Discord webhooks.

## Features

* Multiple shop packs
* Category-based shop menus
* Buy, sell, sell-all, and amount selector menus
* Configurable menu files
* Search menu with chat, sign, and anvil input support
* Multi-currency shops
* Item currency support
* Custom item plugin support
* PlaceholderAPI support
* Favorites and recent transactions
* Sell GUI
* Block-linked shops
* Discord webhooks
* Transaction safety, price checks, and sell limits

## Required Dependencies

None.

ZaminShop can start without Vault, but you need at least one working economy provider before normal money-based shop purchases can work.
