# Permissions

## `zaminshop.player.*`

Grants all player ZaminShop permissions

## `zaminshop.admin.*`

Grants all admin ZaminShop permissions

## `zaminshop.shop`

Legacy alias for zaminshop.player.shop

## `zaminshop.sellgui`

Allows opening the configurable sell GUI

## `zaminshop.admin`

Legacy alias for zaminshop.admin.*

## `zaminshop.admin.sanitize`

Allows removing leaked ZaminShop GUI items from a player inventory
