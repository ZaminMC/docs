# Commands

## /zaminshop

**Description:** Main ZaminShop command

**Aliases:** shop, zshop, zhop

```bash
/<command> [shop|subcommand]
```

## /sell

**Description:** Sell items through ZaminShop

```bash
/<command> <hand|all>
```


# Common Admin Commands

```bash
/zaminshop reload
/zaminshop debug
/zaminshop version
/zaminshop sanitize
```

# Common Player Commands

```bash
/shop
/sell
/shop search <query>
```
