# 🛒 ZaminShop

The modern Minecraft economy shop plugin focused on **performance**, **security**, **customization**, and **clean GUI workflows**.

ZaminShop is not designed to be “just another shop plugin”.  
It is built around:

- ⚡ Fast GUI rendering
- 🛡️ Safe transaction handling
- 🔍 Search-driven navigation
- ❤️ Favorites & recent history
- 🧩 Fully configurable YAML menus
- 📦 1.8 → latest version support
- 🔌 PlaceholderAPI support
- 🧠 Smart GUI architecture
- 🧵 Folia-aware transaction systems
- 🛒 Multi-shop ecosystem

---

# Why ZaminShop?

Most economy plugins focus only on selling items.

ZaminShop focuses on the **entire shopping experience**:

| Feature | Why it matters |
|---|---|
| Search system | Players find items instantly |
| Favorites | Faster repeated purchases |
| Recent history | Quick navigation |
| GUI architecture | Deep customization |
| Central transaction engine | Prevents duplication and race-condition issues |
| Performance modes | Better scalability |
| Security systems | Safer economy handling |

---

# Features

## 🔎 Smart Search
Search items across shops using GUI-based searching.

## ❤️ Favorites
Players can save commonly used items.

## 🕒 Recent Purchases
Recent items are tracked for easier repeat shopping.

## 🧩 Fully Configurable GUIs
Menus are YAML-based and customizable.

## ⚙️ Performance Modes
Tune the plugin for different server sizes.

## 🛡️ Secure Transactions
Atomic transaction architecture designed to reduce duplication issues.

---

# Installation

1. Download the plugin
2. Place the jar into `/plugins`
3. Start the server
4. Edit configuration files
5. Run `/zaminshop reload`
