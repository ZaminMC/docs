# GUI System

The GUI system is one of the core features of ZaminShop.

Menus are YAML-driven and support:

- Dynamic layouts
- Pagination
- Favorites
- Search
- Bulk buy/sell
- Amount selectors

---

# GUI Files

| File | Purpose |
|---|---|
| amount-selector.yml | Amount selection |
| bulk-buy.yml | Bulk purchases |
| bulk-sell.yml | Bulk selling |
| favorites.yml | Favorites GUI |
| recent.yml | Recent purchases |
| search.yml | Search system |
| shop-directory.yml | Shop navigation |

---

# Philosophy

Menus should feel fast, responsive, and modern.
