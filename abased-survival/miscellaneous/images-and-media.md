# Resource Versions

{% hint style="danger" %}
It is crucial to run all server setups using the exact versions specified below. Using earlier versions may cause the world to fail to load, while using later versions may result in certain plugins not functioning correctly. For compatibility with clients on different versions, it is recommended to use ViaVersion and/or ViaBackwards.
{% endhint %}
