# Configuration Overview

ZaminShop configuration is split into purpose-based files.

| File/folder | Purpose |
|---|---|
| `config.yml` | Main plugin behavior, economy providers, safety, logging, sell commands, search, recent menu, database |
| `pricemodifiers.yml` | Stores configured command/rank/player price modifiers |
| `lang/en_US.yml` | Default language messages and placeholder text |
| `guis/*.yml` | Configured inventory menus |
| `shops/*.yml` | Shop categories and shop items |

The plugin also has compatibility loaders for legacy settings, but new docs should point users to the current file layout.
