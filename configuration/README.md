# Configuration

ZaminShop is heavily configuration-driven.

## Main folders

| Folder | Purpose |
|---|---|
| shops/ | Shop definitions |
| guis/ | GUI menus |
| lang/ | Messages |
| config.yml | Main settings |
| pricemodifiers.yml | Price logic |

---

# Best Practice

- Keep backups before editing
- Use YAML validators
- Reload after changes
- Avoid tabs in YAML files
