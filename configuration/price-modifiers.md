# Price Modifiers

Price modifiers allow dynamic pricing.

Examples:

- VIP discounts
- Rank-based pricing
- Event multipliers
- Economy balancing

---

# Use Cases

## VIP Discount

```yaml
vip:
  permission: zaminshop.vip
  multiplier: 0.90
```

## Event Boost

```yaml
event:
  multiplier: 1.20
```
