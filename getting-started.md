# 🚀 Quick Start

## Step 1 — Install Dependencies

Recommended:

- PlaceholderAPI
- Vault
- LuckPerms

Optional economy providers are also supported.

---

## Step 2 — Start the Server

ZaminShop automatically generates:

- config.yml
- shops/
- guis/
- lang/

---

## Step 3 — Open the Shop

Use:

```bash
/shop
```

or

```bash
/zaminshop
```

---

## Step 4 — Customize Shops

Edit files inside:

```text
/plugins/ZaminShop/shops/
```

Each file represents a shop category.

---

## Step 5 — Customize GUIs

GUI files are stored inside:

```text
/plugins/ZaminShop/guis/
```

You can redesign menus entirely.

---

## Step 6 — Reload

```bash
/zaminshop reload
```

Avoid excessive reloads during peak player activity.
