# Integrations

ZaminShop can connect its menus, currencies, and item definitions to other plugins without making those integrations mandatory.

Use this section to configure:

- [PlaceholderAPI](placeholderapi.md) values in menus and messages
- [Economy Providers](economies.md) for money, points, tokens, and other currencies
- [Item and Spawner Hooks](hooks.md) for custom item providers and supported spawner plugins

Only enable integrations that are part of your server stack. Keeping unused providers out of the configuration makes startup diagnostics and transaction behavior easier to understand.
